# Frank Gadgets 🛍️

Frank Gadgets is a modern premium e-commerce store built for buying the latest gadgets — phones, laptops, wearables, accessories and more — with fast delivery across Nigeria.

## About

Frank Gadgets offers a seamless and secure shopping experience with a clean, Apple-inspired design. Customers can browse products, add to cart, checkout, and pay via PalmPay bank transfer or pay on delivery — with instant WhatsApp confirmation.

## Features

- Apple-inspired UI with smooth animations
- Full product catalog with categories and search
- Shopping cart with real-time updates
- Checkout with PalmPay bank transfer + Pay on Delivery
- WhatsApp order confirmation for both payment methods
- Downloadable/printable order receipts
- Full admin panel — orders, products, categories
- Email confirmation when order is accepted
- Mobile responsive across all devices

## Tech Stack

- PHP 8
- MySQL
- Tailwind CSS
- JavaScript (Vanilla)
- PHPMailer (Gmail SMTP)
